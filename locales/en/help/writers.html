Active writers

The writer list is detailed as follow

  • User identifier (Firstname Lastname [1]).
  • Link to modify user's permissions.
  • Permissions' list of writer.

[1]: if the nickname is given it will be used instead of first and last names.

Invite a new writer

User identifier (login): it's the identifier used to login to the blog's administration.

User permissions

To add permissions, you have to check the correspoding checkboxes and to save your selection.

Conversely, to remove permissions, simply uncheck the corresponding boxes and save your selection.

A user who no longer has any permissions on the blog automatically disappears from the list of active editors.

  • manage your own posts and comments: The user has the right to create posts and to access the comments of the posts belonging to him/her.
  • publish posts and comments: Users can change the publication status of their own posts and comments.
  • delete posts and comments: The user can delete his own posts and their comments.
  • manage all posts and comments: The user can fully manage the blog posts, regardless of the author.
  • manage categories: The user can create, modify, delete categories.
  • managing own media: The user can add media and edit or delete his own.
  • manage all media: The user can manage any user's media.
